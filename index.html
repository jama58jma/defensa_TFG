4. Exportar de Blender a Unity

Preparación del modelo 3D para su integración y simulación en tiempo real dentro de Unity.

🧩

Aplicación de modificadores

Se aplicaron modificadores como Mirror y Subdivision para consolidar la geometría final del modelo.

🎨

Bake de materiales

Se realizó el bake de materiales para convertir los nodos y shaders de Blender en texturas compatibles con Unity.

📐

Escala del modelo

Se verificó una escala 1:1 para mantener proporciones correctas durante la simulación física.

📦

Exportación FBX

Finalmente, el modelo fue exportado en formato .fbx para su integración dentro del motor Unity.
